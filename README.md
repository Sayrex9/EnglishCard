# git b


Приложение для изучения английских слов с помощью интерактивных викторин.


## Функционал


* **Авторизация** — вход под своим именем.
* **Викторина** — перевод слов с русского на английский (выбор из 4 вариантов).
* **Добавление слов** — пополнение личного словаря.
* **Удаление слов** — управление личным словарём.
* **Статистика** — прогресс обучения (точность, количество попыток, последние слова).
* **Схема БД** — просмотр структуры базы данных.

## Технологии

* Python 3
* Streamlit — веб‑интерфейс
* PostgreSQL — хранение данных
* psycopg2 — подключение к БД

* random — генерация вариантов ответов

## Запуск

1. Установите зависимости:
   ```
   pip install streamlit psycopg2-binary
   ```

2. Настройте PostgreSQL:

- создайте базу данных english_card;

- убедитесь, что файл init_db.sql (с DDL‑запросами) находится в корневой директории.

3. Запустите приложение:
   ```bash
   streamlit run main.py
   ```
## Структура БД
Приложение использует 4 таблицы:

- ```users``` — пользователи;

- ```common_words``` — базовый словарь;

- ```user_words``` — личные слова пользователя;

- ```learning_stats``` — статистика обучения.

## Конфигурация
В коде жёстко заданы параметры подключения к БД (файл ```main.py```):

- хост: ```localhost```;

- база данных: ```english_card```;

- пользователь: ```postgres```;

- пароль: ```postgres```.
